Controle de Distribuição

Processos abaixo refletem a tabela do termo do dia atual. Atribua um escrevente (obrigatório para auto-distribuição) e, se cabível, um Oficial de Justiça. Use Obs para anotar pendências internas — esse campo não vai para o Teams. Marque OK quando tudo estiver concluído.

Como funciona a distribuição automática:

  • Distribui apenas processos ainda sem escrevente.
  • Equilibra por tipo de processo (cada escrevente fica com aproximadamente o mesmo número de cada classe).
  • Processos do tipo Comunicado de Mandado de Prisão não recebem OJ.
  • Após cada distribuição, os processos atribuídos ficam marcados como “novos” (vão em negrito no Teams). Ao clicar em Limpar “novos”, todos viram regulares.
