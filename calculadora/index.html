Como usar o Gerador em Lote

O Gerador em Lote serve para quando você precisa atualizar vários valores mensais de uma vez — por exemplo, parcelas de aluguel, pensão, restituições mensais etc.

Passo a passo:

  1. Escolha a descrição (ex: Restituição) e o dia do vencimento (ex: dia 10).
  2. Defina se os juros começam a partir de cada vencimento ou de uma data fixa.
  3. Em "Períodos", informe o mês/ano de início, o mês/ano de fim e o valor mensal.
    Ex: 01/2018 a 12/2018 → R$ 500,00
  4. Se o valor muda com o tempo, clique em "+ Adicionar outro período" e preencha o novo intervalo com o novo valor.
    Ex: 01/2019 a 08/2025 → R$ 550,00
  5. Clique em "Gerar Linhas". O sistema cria automaticamente uma linha de cálculo para cada mês do período.

💡 Dica: Após gerar, você pode editar qualquer linha individualmente, caso algum mês específico tenha valor diferente.
